# Iconography

```js
import { Clock } from "neetoicons";
```

Render the `Clock` icon with a specified color and size of 24 units within your
React code.

```jsx
<Clock color="#1e1e20" size={24} />
```

You can access the entire catalog of icons at https://neeto-icons.neeto.com/.
