# What is Formik?

To know about formik, ref the [official documentation](https://formik.org/docs/overview).

<div className="subheading">Installation</div>

Install the formik package in your project.

```
yarn add formik
```

#### Importing

NeetoUI formik exports all itâ€™s component as named exports. You can individually import necessary components in the following way:

```
import { Input } from "@bigbinary/neetoui/formik";
```

Available components in NeetoUI formik :

- Input
- Radio
- Select
- Switch
- Textarea
- CheckBox
- BlockNavigation

#### Acceptable props for Components

Props are similar to that of the basic components and you can refer documentation of each component for a detailed list of avaialble props.
